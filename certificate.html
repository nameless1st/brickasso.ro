Certificate of Authenticity

Each Brickasso® work is accompanied by an official Certificate of Authenticity, individually signed, numbered and linked to the registered work.

The certificate confirms the artwork as an original Brickasso® piece by Albert Oprea and connects it to its unique entry in the official registry.

Each certificate contains a private BRK verification code and secure verification link reserved for the owner. For security and provenance protection, the complete code is never published online.

The certificate records:

Title of the work
Total brick count
Creation details
Signature and date
Registry number and private verification code

"Every brick tells a story, and every certificate preserves it."
Albert Oprea
